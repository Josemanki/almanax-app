# AlmanaxApp

Filtering application for the Dofus Almanax, which makes it very comfortable to check when is the next instance of a specific bonus in the Dofus universe.

The application is made with React and vanilla CSS, data is pulled from a public API.

## How to use

The main part of the application is the filtering function shown in the GIF above: You choose the bonus and it will tell you the next date(s) it will take place on.

In the main screen there's also an overview of the current day and the next 5 days.
